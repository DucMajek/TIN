## Instrukcje Uruchomienia

### Po stronie serwera

1. Wejdź do katalogu serwera:
   ```bash
   cd server
2. Zainstaluj niezbędne zależności:
   ```bash
   npm install

3. Uruchom serwer za pomocą nodemon:
   ```bash
   nodemon index.js

### Po stronie klienta

1. Wejdź do katalogu clienta:
   ```bash
   cd client/tin
2. Zainstaluj niezbędne zależności:
   ```bash
   npm install

3. Uruchom aplikację klienta:
   ```bash
   npm start
